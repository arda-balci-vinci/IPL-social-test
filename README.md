## Description du projet
Ce projet a été réalisé dans le cadre de l’examen de DevOps (BIMV2190).
Il met en place un environnement DevOps avec une pipeline de CI via GitHub Actions.

Le projet utilise la méthodologie TDD (Test Driven Development) pour valider
une fonction JavaScript de vérification de mot de passe.  
Les tests sont automatiquement exécutés lors d’une Pull Request vers la branche
principale `main`.
